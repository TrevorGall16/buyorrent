3. UI/UX SPECIFICATIONS (v2.0)

## 1. Design System (Custom Tailwind)
* **Theme:** "Modern Fintech."
* **Background:** Radial gradient grid (`bg-slate-50` to `white`).
* **Cards:** Custom CSS classes. White background, `border-slate-200`, `shadow-sm`.
* **Primary Color:** Blue (#3b82f6).
* **Semantic Colors:** Green (#10b981) for Buy, Red (#ef4444) for Rent.

## 2. Component Inventory
* **Buttons:** Custom `Button.tsx`. rounded-full, gradient bg, hover elevation.
* **Inputs:** Custom range sliders using standard `<input type="range" />` with Tailwind accent colors.
* **Cards:** Simple `div` wrappers with `rounded-2xl` and `border`.

## 3. Visual Polish (Anti-AI Rules)
* **Typography:** System fonts. Headings = `tracking-tight`. Labels = `text-sm`, `font-medium`, `text-slate-600`.
* **Micro-Interactions:** Buttons `scale-95` on click. Inputs `ring-2` on focus.
* **Depth:** Cards need a 1px top highlight (`border-t border-white/60`) to simulate light.

## 4. Internationalization UI
* **Formatting:** Currency symbols (€/$/£) must match the selected country config.
