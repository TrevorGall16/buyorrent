2. Country Config (lib/country-config.ts)

Must map country_code to financial rules:

    DE (Germany): Closing Cost = 12% (Kaufnebenkosten).

    FR (France): Closing Cost = 7.5% (Notaire).

    US (USA): Closing Cost = 3%.
