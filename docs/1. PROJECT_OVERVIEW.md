# 1. PROJECT OVERVIEW: RentOrBuy-Pro (v2.0)

## 1. Project Context
* **Type:** Programmatic SEO Financial Tool (International).
* **Core Purpose:** To provide a precise, math-backed "Rent vs. Buy" financial verdict for 46+ cities across 14 countries.
* **Core Value:** "Speed is Trust." Instant calculations, local tax laws, and zero-fluff answers.

## 2. Target Users
* **Primary:** First-time homebuyers and expats trying to decide on property in specific cities (e.g., "Rent vs Buy in Berlin").
* **Intent:** High-intent financial research. They want numbers, not opinion.

## 3. Core Features (Input → Output)
* **Calculator Engine:** Inputs (Price, Rent, Tax, Duration) → Output (Net Worth Comparison & Break-Even Year).
* **International Logic:** Country Selection → Swaps Tax/Closing Cost Logic (e.g., 12% costs in DE vs 3% in US).
* **Programmatic Pages:** URL (`/paris/buy-vs-rent`) → Pre-filled City Data.
* **Global Mode:** Region Selector → Custom calculation for non-database cities.

## 4. Constraints & Priorities
* **Performance:** **INP < 100ms** (Interaction to Next Paint).
* **Indexing:** 100% of City Pages must be indexed (SSG).
* **Revenue:** CLS-Safe AdSense integration (Zero Layout Shift).
* **Tech:** Strict Next.js 16 + React 19 + Shadcn/UI architecture.

## 5. Success Criteria
* **Speed:** 95+ Lighthouse Performance Score.
* **Reach:** 50+ Unique landing pages indexed by Google.
* **Trust:** Math accuracy validated against local tax laws (Verified via Audit).
